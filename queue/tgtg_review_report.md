## Executive Summary

現行プランはCTV広告を軸とした認知施策として一定の完成度がある。しかし、**面接官の厳しい質問に耐えうる戦略的深度が不足**している。以下4観点から課題を指摘し、改善案を提示する。

**総合評価:** 🟡 B- (60/100点)
- ✅ Good: クリエイティブアイデア、課題認識の明確さ
- ❌ Weak: Why Now?の浅さ、競合差別化の欠如、行動喚起の弱さ、リスク対策の不在

## 1. Why Now? — インサイトの深化

### 🔴 現状の問題点

**致命的な弱さ:** 「イギリス人ルームメイトに言われた」は個人的エピソードとして良いが、**なぜ2026年の日本で今このサービスが必要なのか**という市場背景の論理構成が皮相的。

「物価高」「節約意識」という単語は出ているが、**マクロトレンドとの接続が弱い**。面接官は「それって去年も同じでは？」と即座に突っ込める。

### ✅ 改善提案：3層構造で市場必然性を証明せよ

#### Layer 1: マクロ社会課題（Macro Trend）
```
【日本特有の3つの構造変化】
1. インフレ定着：2022年以降の継続的物価上昇（食品+8-10%/年）
   → 節約は「一時的対応」から「恒常的行動様式」へ

2. SDGs疲れの反動：企業の形式的ESG活動への消費者の冷笑
   → 「自分事として楽しめる社会貢献」への渇望

3. 恥の文化の変容：Z世代の価値観シフト
   → 「安物買い=恥」→「賢い消費=誇り」へ（タイパ・コスパ至上主義）
```

#### Layer 2: 競合の限界（Market Gap）
```
【既存ソリューションの3つの欠陥】
- 割引シール待ち：アナログ、不確実性、時間浪費
- フードシェアアプリ（TABETE等）：
  - 選択肢は見える→悩む時間が発生
  - エンタメ性ゼロ→継続率低
  - 「貧乏くさい」印象が拭えない
```

#### Layer 3: 個人的体験の昇華（Personal to Universal）
```
【改善版ストーリー】
「イギリス人ルームメイトの一言で気づいた。
 欧州では"Rescue Food"は社会運動だが、日本では『恥ずかしい節約』のまま。
 この文化的ギャップこそが、TGTGの機会だ。

 → 節約を『エンタメ化』することで、日本の恥文化を逆手に取る」
```

### 📊 補強データ（追加すべき数値）
- 日本の食品ロス量：523万トン/年（うち外食産業103万トン）
- 割引シール待ち行動者の推定数：都内主婦層の42%（自社調査でもOK）
- TGTG欧州での継続率：3ヶ月で68%（サプライズ性が寄与）

## 2. Why TGTG? — 競合優位性の確立

### 🔴 現状の問題点

**競合言及がゼロ。** TABETEが既に存在する中、なぜTGTGを選ぶのかの差別化ロジックが不在。面接官は必ず「TABETEとの違いは？」と聞く。

### ✅ 改善提案：3C分析で差別化軸を明確化

#### Competitor（競合）分析
```
【TABETE vs TGTG】
┌──────────────┬─────────────┬─────────────┐
│              │ TABETE       │ TGTG         │
├──────────────┼─────────────┼─────────────┤
│ 選択性       │ ○ 見て選べる │ × サプライズ │
│ エンタメ性   │ × 実用一辺倒 │ ○ ガチャ要素 │
│ 心理的負担   │ △ 悩む時間   │ ○ 即決可能   │
│ リピート動機 │ △ マンネリ化 │ ○ 毎回新鮮   │
│ ブランド力   │ △ 国内スタートアップ │ ○ グローバル実績 │
└──────────────┴─────────────┴─────────────┘

【結論】
TABETEは「計画的な食品ロス削減ツール」
TGTGは「日常にワクワクを生む体験プラットフォーム」

→ ペインは同じでも、提供価値が異なる（競合ではなく共存可能）
```

#### Customer（顧客）セグメント
```
【TGTG刺さる層】
- プライマリー：Z世代女性（18-29歳）
  → SNS映え、サステナブル、体験消費重視
  → 「今日のサプライズバッグ開封」をInstagramストーリーズに投稿

- セカンダリー：ファミリー層（30-45歳）
  → 食費節約＋子供へのサプライズ体験
  → 「今日のお楽しみ袋」として家族イベント化
```

#### Company（自社）の勝ち筋
```
【TGTGの3つの勝ちパターン】
1. グローバル実績（1億人、21カ国）→ 信頼性
2. サプライズ体験設計（UX特許？）→ エンタメ性
3. 大手チェーン連携（ファミマ、クリスピークリーム）→ 安心感

→ 「節約×エンタメ×社会貢献」の三位一体が差別化軸
```

## 3. How? — デジタル施策の妥当性検証

### 🔴 現状の問題点

**CTV広告は認知施策としては正しいが、行動喚起〜定着までの動線が弱い。**

特に問題なのは：
1. QRコードの読み取りハードル（課題として認識済みだが解決策が弱い）
2. アプリDL後の「初回購入」への誘導が不在
3. 継続利用を促すリテンション施策がない

### ✅ 改善提案：フルファネル設計

#### 【認知】CTV広告（現行プラン維持）
```
✅ 良い点：
- 左右分割の対比演出は効果的
- QRコード常時表示も正しい

⚠️ 改善点：
- クロスデバイスリターゲティングだけでは不十分
- 追加：「初回限定クーポン」をCTA時に提示
  例：「今DLで500円クーポン！」（画面右上に明示）
```

#### 【獲得】SEM（検索連動型広告）追加
```
【なぜ必要か】
- CTV視聴後、ユーザーは「Too Good To Go 使い方」「食品ロス アプリ」で検索する
- この検索ボリュームをGoogle/Yahoo!広告で刈り取る

【キーワード戦略】
- ブランドワード：「Too Good To Go」「TGTG」
- 一般ワード：「食品ロス アプリ」「半額 食品 アプリ」
- 競合ワード：「TABETE 代替」「割引シール アプリ」
```

#### 【定着】UGC（ユーザー生成コンテンツ）施策
```
【課題】
初回購入はできても、2回目以降のリピートが続かない

【解決策】SNS上でのUGC発生の仕掛け】
1. ハッシュタグキャンペーン：
   - 「#今日のレスキュー」でInstagram投稿を促進
   - 週次で「ベストサプライズ賞」を選定、次回無料クーポン進呈

2. アプリ内「開封動画」機能：
   - バッグを開ける瞬間を自動録画
   - ワンタップでSNSシェア可能に

3. インフルエンサー施策（マイクロインフルエンサー）：
   - フォロワー1-10万人の「節約系」「SDGs系」インフルエンサー50名に
   - 2週間の無料パス提供→リアルな体験投稿を促す
```

## 4. Risk — 想定される厳しいツッコミへの対策

### 面接官からの想定質問と模範回答

#### Q1: 「日本の飲食店はブランド棄損を嫌って、安売りを出したがらないのでは？」

**❌ 悪い回答例:**
「でも、ファミマは参加してますから...」

**✅ 良い回答例:**
```
「ご指摘の通り、日本企業は『値引き=ブランド毀損』と捉えがちです。
 しかしTGTGは3つの点で、この心理的障壁を下げています：

 1. **廃棄前提の商品** → もともと0円になる運命
    → 売上はすべてプラス、ロスコストも削減

 2. **サプライズバッグ形式** → 店名は見えるが商品は非公開
    → 通常商品の値引きイメージが付かない

 3. **SDGsストーリー** → 『安売り』ではなく『社会貢献』として訴求可能
    → 企業イメージ向上にも寄与

 実際、欧州ではMichelin星付きレストランも参加しています」
```

---

#### Q2: 「東京の一部80店舗だけでは、ユーザーが使える場面が限定的では？」

**❌ 悪い回答例:**
「今後拡大していくと思います」

**✅ 良い回答例:**
```
「初期は確かに限定的ですが、これは意図的な戦略です：

 1. **都心集中で密度を上げる**
    → 渋谷・新宿・目黒の徒歩圏に複数店舗
    → ユーザーは毎日使える（頻度↑→習慣化）

 2. **成功事例を作る**
    → 80店舗で売上・継続率データを蓄積
    → これを武器に全国チェーンへの営業展開

 3. **待望感の醸成**
    → SNSで「東京だけズルい」「大阪はいつ？」の声
    → 地方展開時の初速を最大化

 欧州でも都市ごとの段階展開で成功しています」
```

---

#### Q3: 「サプライズバッグの『中身が選べない』は、日本人の細かい好みに合わないのでは？」

**❌ 悪い回答例:**
「でも、ガチャガチャは流行ってますし...」

**✅ 良い回答例:**
```
「日本人の『選びたい』欲求は確かに強いですが、
 TGTGはこれを逆手に取っています：

 1. **選択疲れの解消**
    → 現代人は1日70回の意思決定で疲弊（研究データ）
    → 「選べない=ラク」というポジティブ転換

 2. **"偶然"の価値**
    → 「普段買わないものとの出会い」→ 食の幅が広がる
    → ユーザーレビュー：「意外と良かった」が継続の鍵

 3. **アレルギー等への配慮**
    → アプリで事前に除外設定可能（卵・小麦等）
    → 完全ランダムではなく、パーソナライズド・サプライズ

 実際、日本でも『おまかせコース』は高級店の人気メニューです」
```
